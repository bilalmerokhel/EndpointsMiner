# How To
- Install jq first `https://stedolan.github.io/jq/download/`
- `git clone https://github.com/bilalmerokhel/EndpointsMiner.git`
- `cd EndpointsMiner/`
- `chmod +x EndpointsMiner.sh`
- `./EndpointsMiner.sh`
